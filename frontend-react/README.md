# Frontend (Vite + React + TypeScript)

Минималистичный интерфейс для сокращения ссылок.

## Команды

- `npm install` — установить зависимости
- `npm run dev` — запустить dev-сервер (по умолчанию на `http://localhost:5173`)
- `npm run build` — сборка production в папку `dist`
- `npm run preview` — локальный предпросмотр production-сборки

## API

Фронтенд отправляет запросы в `http://localhost:8000/api/reduce`.
